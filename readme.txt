Built with React, Express & MongoDB

In this project I use mongodb. Have a look at .envExample for the required env variables.
1. Open terminal in root folder
2. Run "npm install"
3. Navigate to /client
4. Run "npm install" and after installation, run "npm start"
5. Open another terminal window
6. Navigate to /server
7. Run npm install and after installation, run "npm start" (nodemon)
